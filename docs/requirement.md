# 自動出欠管理アプリ　要件定義書

## 画面要件
- ログイン画面
- メイン画面(ダッシュボード)
- 学校設定画面
- 授業管理画面
- 生徒管理画面
- 履修登録機能
- 出欠管理画面

## 画面別機能要件

### ログイン
- ログイン機能

### メイン画面
- 学年別の出席率を表示

### 学校管理画面
- 教室CRUD操作機能
- 時間枠CRUD操作機能

### 授業管理画面
- 授業CRUD操作機能

### 生徒管理画面
- 生徒CRUD操作機能
    1. クラスごとに生徒を追加することができるようにする
    2.  .csvファイルでも生徒を追加できるようにする
    3. .csvは 生徒ID, 学年, 組, 出席番号がセットになった情報が記録されています
    4. 生徒の所属するコース（専攻）なども追加できるようにする

### 履修登録画面
- 各授業をドロップダウンで選択肢を表示
- ドロップダウンで選択された授業で授業を履修する生徒のCRUD操作機能

### 出欠管理画面
- 各授業をドロップダウンで選択肢を表示
- 日付を選択するボタン, カレンダーを表示
- ドロップダウンで選択された条件に合う出席情報を表示
